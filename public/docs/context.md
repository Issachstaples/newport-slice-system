# Newport Slice System – Context Anchor

## Project
- Goal: Build a reusable Prismic slice library for SaaS + agency sites
- Stack: Next.js (App Router), Tailwind, Prismic Slice Machine v2
- Design language: Apple-style liquid glass, SaaS-dense layouts
- Reuse-first philosophy (minutes not weeks)

## Slices Built
- HeroSystem
  - Badge pill with pulsing dot
  - Visual modes (gradient_orb, helix_3d, glass)
- BentoGrid
  - SaaS-dense layout
  - Real icons (lucide-react)
  - Pulse only on highlighted cards
- CtaSection
  - Footer-style closer

## Visual Rules
- Subtle but readable glass
- Dense spacing, not marketing fluff
- Highlight = pulse
- Icons live inside frosted chips
